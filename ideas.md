## 아이디어 1: "Moonlit Workspace" — 달빛 아래 작업실

**Design Movement**: Scandinavian Minimalism + Warm Industrial
**Core Principles**:
- 따뜻한 뉴트럴 톤 위에 골드 악센트가 빛나는 구조
- 기능 중심의 대시보드, 불필요한 장식 최소화
- 넓은 여백과 명확한 정보 위계

**Color Philosophy**: 
- 배경: 따뜻한 오프화이트(#FAFAF8) + 라이트 웜그레이(#F0EFED)
- 텍스트: 차콜(#2D2D2D) + 미디엄그레이(#787878)
- 악센트: 달빛워크 골드(#E9A820) — CTA, 중요 수치, 호버에만 사용
- 테이블/카드: 화이트 + 미세한 보더(#E5E5E3)

**Layout Paradigm**: 
- 좌측 네비게이션 사이드바(접힘 가능) + 우측 넓은 작업 영역
- 견적서 작성은 카드 기반 폼, 미리보기는 A4 비율 프레임

**Signature Elements**:
- 골드 컬러 언더라인이 있는 섹션 타이틀
- 미세한 그림자와 1px 보더로 깊이감 표현
- 숫자/금액에 모노스페이스 폰트 적용

**Interaction Philosophy**: 
- 항목 추가/삭제 시 부드러운 슬라이드 애니메이션
- 금액 변경 시 숫자 카운팅 트랜지션
- 호버 시 골드 악센트 강조

**Animation**: 
- 페이지 전환: 페이드 + 약간의 Y축 이동 (200ms)
- 카드/행 추가: 슬라이드 다운 + 페이드인
- 버튼 호버: 배경색 전환 (150ms ease)

**Typography System**: 
- 제목: Pretendard Bold (700)
- 본문: Pretendard Regular (400)
- 금액/숫자: JetBrains Mono 또는 Roboto Mono
- 크기 체계: 32/24/18/15/13px

## 아이디어 2: "Dark Forge" — 어둠 속 대장간

**Design Movement**: Dark Mode Professional + Neo-Brutalism 요소
**Core Principles**:
- 다크 배경 위에 골드가 강렬하게 빛나는 프리미엄 느낌
- 견적서 = 무기를 벼리듯 정밀하게 다듬는 도구
- 굵은 보더와 선명한 대비

**Color Philosophy**: 
- 배경: 딥 차콜(#1A1A1A) + 다크그레이(#252525)
- 텍스트: 라이트그레이(#E0E0E0) + 화이트(#FFFFFF)
- 악센트: 골드(#E9A820) + 앰버(#D4941A)
- 카드: 약간 밝은 다크(#2A2A2A) + 골드 보더

**Layout Paradigm**: 
- 풀스크린 단일 컬럼, 스크롤 기반
- 견적서 작성 영역이 화면 중앙에 A4 용지처럼 떠있는 형태
- 상단에 미니멀한 탑바

**Signature Elements**:
- 골드 글로우 이펙트가 있는 총액 표시
- 2px 솔리드 보더의 네오브루탈리즘 카드
- 다크 배경 위 골드 타이포그래피

**Interaction Philosophy**: 
- 클릭/탭 시 골드 리플 이펙트
- 드래그로 항목 순서 변경
- 다크 모드 특유의 미세한 글로우 호버

**Animation**: 
- 요소 등장: 스케일(0.95→1) + 페이드 (300ms)
- 금액 변경: 글로우 펄스 효과
- 삭제: 좌측으로 스와이프 아웃

**Typography System**: 
- 제목: Space Grotesk Bold
- 본문: Inter Medium
- 금액: Space Mono Bold
- 크기 체계: 36/28/20/16/14px

## 아이디어 3: "Paper & Ink" — 종이와 잉크

**Design Movement**: Swiss Design (International Typographic Style) + Stationery Aesthetic
**Core Principles**:
- 실제 종이 견적서의 질감과 구조를 디지털로 재해석
- 그리드 시스템에 충실한 정보 배치
- 흑백 + 골드의 3색 체계 엄격 준수

**Color Philosophy**: 
- 배경: 순백(#FFFFFF) + 페이퍼 그레이(#F5F5F5)
- 텍스트: 퓨어 블랙(#1A1A1A) + 미디엄그레이(#888888)
- 악센트: 골드(#E9A820) — 극히 제한적으로 사용 (버튼, 총액, 상태 뱃지)
- 구분선: 라이트그레이(#E0E0E0) + 블랙(#323232)

**Layout Paradigm**: 
- 중앙 정렬 단일 컬럼 (max-width: 900px)
- 견적서 작성 = 실제 A4 용지 위에 타이핑하는 느낌
- 목록 페이지는 테이블 기반의 스프레드시트 느낌

**Signature Elements**:
- 실제 견적서 PDF와 거의 동일한 웹 레이아웃 (WYSIWYG)
- 얇은 헤어라인 구분선으로 섹션 분리
- 타이프라이터 느낌의 모노스페이스 숫자

**Interaction Philosophy**: 
- 최소한의 인터랙션, 입력에 집중
- 탭 키로 빠르게 필드 이동
- 인라인 편집 (클릭하면 바로 수정)

**Animation**: 
- 거의 없음, 있더라도 100ms 이내의 미세한 전환
- 저장 시 체크마크 아이콘만 잠깐 표시
- 프린트 미리보기 전환 시 종이 펼치는 듯한 스케일 애니메이션

**Typography System**: 
- 제목: Noto Sans KR Bold (700)
- 본문: Noto Sans KR Regular (400)  
- 금액/숫자: IBM Plex Mono
- 크기 체계: 28/22/16/14/12px
